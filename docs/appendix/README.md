## Contributing to Appendix

### Adding Screenshots

1. Take screenshot at 1920x1080 or higher
2. Annotate important UI elements
3. Save as PNG in `docs/assets/screenshots/`
4. Add to [Screenshots](screenshots.md) with description

### Adding Videos

1. Record screen at 1080p
2. Keep videos under 5 minutes
3. Upload to YouTube or Vimeo
4. Add link to [Video Tutorials](video-tutorials.md)

### Updating Changelog

When releasing new version:
1. Document all changes
2. Categorize (features, fixes, breaking)
3. Add to [Changelog](changelog.md)
4. Use semantic versioning

### Adding Resources

Found useful resource?
1. Verify link is official/authoritative
2. Add to appropriate category in [Resources](resources.md)
3. Include brief description
